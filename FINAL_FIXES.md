## 📋 Исправленные проблемы

### 1. **Правильное определение координат при переключении режимов** 🟢

#### Проблема
При включении IP режима - координаты не обновлялись (использовались старые).
При переключении на GPS - не запрашивались свежие координаты.

#### Решение
Улучшена логика в `onReceive` для отслеживания переключения режимов:

```swift
.onReceive(locationManager.$useIPLocation) { useIP in
    Task {
        if useIP {
            // Переключились на IP - запросим IP координаты
            await locationManager.fetchIPLocation()
        } else if locationManager.isAuthorized {
            // Переключились на GPS - запросим GPS координаты
            locationManager.requestLocation()
            try? await Task.sleep(nanoseconds: 500_000_000)  // 0.5 сек
        }
        await loadWeather()
    }
}
```

#### Результат
- ✅ **IP режим**: Вызывается `fetchIPLocation()` → получаются свежие IP координаты → загружается погода
- ✅ **GPS режим**: Вызывается `requestLocation()` → ждёт GPS → загружается погода с GPS координатами

### 2. **Кнопка "Обновить" всегда запрашивает свежие координаты** 🟢

#### Было
При нажатии "Обновить" в IP режиме использовались кэшированные координаты (если были).

#### Стало
```swift
private func loadWeather() async {
    if locationManager.useIPLocation {
        // IP режим: ВСЕГДА запросим свежие IP координаты
        await locationManager.fetchIPLocation()
        guard let ipLoc = locationManager.ipLocation else {
            errorMessage = localization.localize("no_location")
            return
        }
        latitude = ipLoc.latitude
        longitude = ipLoc.longitude
    } else {
        // GPS режим: ВСЕГДА запросим свежие GPS координаты
        if locationManager.isAuthorized {
            locationManager.requestLocation()
            // Ждем до 2 секунд получения GPS
            var gpsLocation: CLLocationCoordinate2D?
            for _ in 0..<20 {
                if let location = locationManager.location {
                    gpsLocation = location
                    break
                }
                try? await Task.sleep(nanoseconds: 100_000_000)
            }
            // Используем полученные координаты
        }
    }
}
```

#### Результат
- ✅ **IP режим**: При нажатии "Обновить" → свежий запрос к ipapi.co → новые IP координаты → новая погода
- ✅ **GPS режим**: При нажатии "Обновить" → новый запрос GPS → новые координаты → новая погода
- ✅ **Без задержек**: Максимум 2 секунды на получение GPS, IP загружается быстро
