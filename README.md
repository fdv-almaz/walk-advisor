# Walk Advisor 🚶

Нативное приложение для macOS, которое помогает вам решить, идти ли гулять или заниматься активными видами спорта на основе данных о погоде и вашей локации.

## ✨ Основные возможности

- 🌍 **Умное определение локации** — GPS или IP-геолокация с автоматическим переключением
- 🌤️ **Текущая погода** — температура, влажность, скорость ветра в реальном времени
- 📊 **Индекс активности** — интеллектуальная оценка (0-100%), показывающая, идеально ли идти гулять
- 📅 **Прогноз на 5 дней** — планируйте прогулки на неделю вперед
- 🌐 **4 языка** — English, Русский, Polski, Беларускі
- 🎨 **Красивый интерфейс** — минималистичный дизайн с цветовой кодировкой
- ⚡ **Нет зависимостей** — чистый Swift с использованием только системных фреймворков
- 🔒 **Приватность** — не требует интернет-ключей, работает полностью локально (кроме API погоды)

## 🚀 Быстрый старт

### Требования

- macOS 12.0 или выше
- Для разработки: Swift 5.5+, Xcode 14+ (опционально)

### Установка и запуск

**Способ 1: Через скрипт (самый быстрый)**

```bash
open build/Walk\ Advisor.app
```

```bash

./build_universal.sh
open build
```

### Первый запуск

1. Запустите приложение
2. На запрос разрешения локации выберите "Allow"
3. Приложение загрузит данные о погоде для вашего региона
4. Просмотрите рекомендацию активности на сегодня

> **💡 Если GPS не работает:** Нажмите кнопку переключения "IP" для использования IP-геолокации

## 📊 Как работает индекс активности?

Приложение анализирует три фактора:

| Фактор | Вес | Оптимально | Хорошо | Плохо |
|--------|-----|-----------|--------|--------|
| **Температура** | 40% | 15-25°C | 10-30°C | <10°C или >35°C |
| **Погода** | 40% | ☀️ Ясно | ☁️ Облачно | 🌧️ Дождь, ⛈️ Гроза |
| **Ветер** | 20% | <15 км/ч | <20 км/ч | >25 км/ч |

**Результаты:**
- 🟢 **80-100%** — Идеально идти гулять!
- 🔵 **60-79%** — Хорошие условия, можно выходить
- 🟠 **40-59%** — Неплохо, но есть трудности
- 🔴 **<40%** — Лучше остаться дома

## 🔧 Разработка

### Требования для разработки

```bash
# Проверьте версию Swift
swift --version

# Проверьте наличие swiftc
swiftc -version
```

### Изменение кода

1. **Добавление нового языка:**
   - Отредактируйте `Localization.swift` и добавьте новый язык в словари
   - Обновите UI для переключателя языков в `ContentView.swift`

2. **Модификация UI:**
   - Отредактируйте `ContentView.swift`
   - Все строки должны использовать `localization.localize("key")`

3. **Изменение алгоритма активности:**
   - Отредактируйте `ActivityScore.init()` в `Models.swift`
   - Измените веса компонентов (температура 40%, погода 40%, ветер 20%)

4. **Смена API погоды:**
   - Отредактируйте `WeatherService.swift`
   - Обновите структуры данных в `Models.swift` если нужно

### Компиляция после изменений

```bash
cd WeatherApp && ./build.sh && open build/Walk\ Advisor.app
```

### Тестирование

```bash
# Быстрое тестирование текущей архитектуры
./test_app.sh

# Тестирование обеих архитектур (Intel + Apple Silicon)
cd WeatherApp && ./build_universal.sh
open build/Walk\ Advisor.arm64.app  # Apple Silicon
open build/Walk\ Advisor.x86_64.app # Intel
```

## 🌐 Используемые API и сервисы

### Open-Meteo Weather API
- **URL:** https://open-meteo.com/
- **Лицензия:** Open-source
- **Особенности:**
  - Бесплатен, без API ключа
  - Открытый источник (GNU AGPL v3)
  - Не требует регистрации
  - Охватывает весь мир

### IP Геолокация (ipapi.co)
- **URL:** https://ipapi.co/
- **Использование:** Резервная геолокация, если GPS недоступен
- **Приватность:** Только определение города по IP

## 🐛 Известные проблемы и их решение

### GPS не работает
- Проверьте разрешение в System Settings > Privacy & Security > Location Services
- Попробуйте IP-геолокацию (кнопка "IP" в приложении)

### Погода не обновляется
- Проверьте интернет соединение
- Убедитесь, что Open-Meteo API доступен (https://open-meteo.com/)

### Неправильный город определяется
- Это может быть связано с IP-геолокацией (обычно неточна на ±20 км)
- Используйте GPS для более точного определения
