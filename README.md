# Walk Advisor 🚶‍♂️

Нативное приложение для macOS, которое помогает вам решить, идти ли гулять или бегать. Анализирует текущую погоду и рекомендует уровень активности на основе температуры, условий и скорости ветра.

![macOS](https://img.shields.io/badge/macOS-12.0%2B-green)

## ✨ Основные функции

- 🌍 **Определение местоположения** — GPS или IP-адрес (автоматический выбор)
- 📊 **Рекомендации активности** — от 0 до 100% (зеленый, синий, оранжевый, красный)
- 📅 **Прогноз на 5 дней** — планируйте неделю заранее
- ℹ️ **About Dialog** — информация о приложении и разработчике (v2.0.2)
- 🌐 **4 языка** — английский, русский, польский, беларусский
- 🎨 **Красивый дизайн** — минималистичный интерфейс SwiftUI
- ⚡ **Оптимизирован** — ~40 MB памяти, быстрая загрузка
- 🔒 **Приватный** — без отслеживания, данные остаются на вашем компьютере

## 🚀 Быстрый старт

### Требования

- **macOS 12.0 или выше** (Monterey, Ventura, Sonoma)
- **Intel или Apple Silicon** (все поддерживаются автоматически)

### Установка

1. **Скачайте приложение:**
   - Распакуйте `Walk Advisor_2.0.2.dmg` из папки `dist/`
   - Перетащите `Walk Advisor.app` в папку **Applications**

2. **Запустите приложение:**
   - Откройте **Applications → Walk Advisor**
   - Разрешите доступ к геопозиции при первом запуске

3. **Готово!** Приложение автоматически покажет рекомендации активности

### Установка из исходников

```bash
cd WeatherApp
./build.sh
open build/Walk\ Advisor.app
```

## 🔧 Компоненты

### Swift исходники
- **WeatherApp.swift** — точка входа приложения, конфигурация окна
- **ContentView.swift** — основной интерфейс (SwiftUI)
- **Models.swift** — структуры данных и алгоритм расчета активности
- **LocationManager.swift** — GPS и IP геопозиция с fallback
- **WeatherService.swift** — интеграция с Open-Meteo API
- **Localization.swift** — многоязычная поддержка (i18n)

### API
- **Open-Meteo** — бесплатный API прогноза погоды (без ключа)
- **ipapi.co** — определение местоположения по IP-адресу

## 🎯 Алгоритм рекомендаций

Приложение рассчитывает уровень активности (0-100%) на основе:

- **Температура (40%)** — идеально 15-25°C
- **Погода (40%)** — ясно > облачно > дождь > гроза
- **Ветер (20%)** — идеально <15 км/ч

**Цветовая схема:**
- 🟢 80-100% — Идеально для прогулки
- 🔵 60-79% — Хорошо для прогулки
- 🟠 40-59% — Допустимо
- 🔴 0-39% — Лучше остаться дома

## 🐛 Известные проблемы и решения

### GPS не работает
```bash
# Дайте разрешение на геопозицию
tccutil grant location /Applications/Xcode.app
```

### Неправильное определение города
- IP-геопозиция имеет точность на уровне города
- Для точного результата дайте разрешение на GPS

### Приложение медленно загружает погоду
- Проверьте интернет соединение
- Убедитесь, что api.open-meteo.com доступен

Больше ответов в [CLAUDE.md](CLAUDE.md) в разделе "Troubleshooting".
