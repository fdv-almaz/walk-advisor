# Walk Advisor - История обновлений

## Версия 2.0 - Большое обновление (2026-05-22)

### ✨ Новые функции

#### 1. GPS / IP выбор
- ✅ Автоматический запрос использования IP если GPS недоступна
- ✅ Диалоговое окно (не запоминает выбор)
- ✅ Переключатель GPS/IP в основном интерфейсе
- ✅ Сохранение выбора в UserDefaults (запоминает при перезагрузке)
- 🌐 Поддержка определения по IP через ipapi.co

#### 2. Прогноз на 5 дней
- ✅ Добавлена интеграция с daily forecast API
- ✅ Отображение прогноза на 5 дней в карточках
- ✅ Показ макс/мин температуры
- ✅ Тип погоды для каждого дня
- ✅ Красивое форматирование дат

#### 3. Вертикальная форма окна
- ✅ Размер окна: 400x700 пикселей
- ✅ Минимальный размер: 380x600
- ✅ Максимальная высота: без ограничения (scrollable)
- ✅ Вертикальная ориентация всех элементов

#### 4. Уровень активности в процентах
- ✅ Замена дискретного вердикта на процентную шкалу (0-100%)
- ✅ Динамический расчет по формуле:
  - Температура: 40%
  - Погода: 40%
  - Ветер: 20%
- ✅ Цветная дифференциация:
  - 80-100%: Зеленый (Идеально)
  - 60-79%: Синий (Хорошо)
  - 40-59%: Оранжевый (Допустимо)
  - 0-39%: Красный (Плохо)

#### 5. Иконка приложения
- ✅ Автоматическое создание набора иконок (16-512px)
- ✅ Использование встроенной иконки macOS Maps
- ✅ Отображение в Finder, Dock, Spotlight
- ✅ Скрипт create_icon.sh для управления

#### 6. Проверка размещения элементов
- ✅ ScrollView для предотвращения обрезания контента
- ✅ Правильные отступы и padding
- ✅ Масштабируемые элементы
- ✅ Адаптивная высота контейнеров
- ✅ Проверенные максимальные размеры на iPhone/Mac

### 🔧 Технические изменения

#### Обновленные файлы

**Models.swift**
- Добавлен `DailyForecast` для прогноза на несколько дней
- Добавлен `DayForecast` для отдельного дня
- Заменен `ActivityRecommendation` на `ActivityScore` класс
- Добавлены методы расчета с процентами
- Новые цветовые схемы

**LocationManager.swift**
- Добавлена поддержка IP определения через ipapi.co
- Добавлены методы `fetchIPLocation()` и `toggleLocationSource()`
- Добавлено сохранение выбора в UserDefaults
- Добавлена структура `IPResponse` для парсинга

**WeatherService.swift**
- Обновлен запрос API для получения daily forecast
- Заменен метод `getActivityRecommendation` на `getActivityScore`
- Добавлен метод `getForecastDays()` для обработки прогноза
- Оптимизирована формула расчета рекомендаций

**ContentView.swift** (полная переработка)
- Новая вертикальная архитектура
- Добавлены представления:
  - `currentWeatherView` - текущая погода
  - `scoreView` - уровень активности с прогресс-баром
  - `forecastView` - прогноз на 5 дней
  - `controlsView` - переключатель и кнопки
- Добавлена ScrollView для защиты от обрезания
- Добавлены методы форматирования дат
- Поддержка IP определения с диалогом

**WeatherApp.swift**
- Увеличены размеры окна (400x700)
- Добавлена инициализация окна
- Оптимизирован стиль окна

**Info.plist**
- Добавлено `CFBundleIconFile` для иконки
- Обновлен `CFBundleName` на "Walk Advisor"
- Обновлен `CFBundleIdentifier` на "com.walkadvisor.macos"

**build.sh**
- Добавлено копирование иконки в приложение
- Оптимизирован вывод информации
