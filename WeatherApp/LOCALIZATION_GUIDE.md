## 🗺️ Система определения местоположения

### GPS определение

**Процесс**:
1. Запрашивает разрешение на доступ
2. Получает координаты (широта/долгота)
3. Использует reverse geocoding для получения города
4. Отображает "Город, Страна"

**Точность**: ±10 метров

### IP определение

**Процесс**:
1. Запрашивает данные у ipapi.co
2. Получает город и страну из IP
3. Отображает "Город, Страна"

**Точность**: ±100 км

### Переключение

- Используйте toggle в интерфейсе
- Автоматически переключит источник
- Место обновится после переключения
- Выбор сохранится в UserDefaults

## 💾 Сохранение настроек

### Что сохраняется

1. **Язык интерфейса**
   - Ключ: `currentLanguage`
   - Сохраняет: языковой код (en, ru, pl, be)

2. **Источник GPS/IP**
   - Ключ: `useIPLocation`
   - Сохраняет: boolean (true/false)

### Где сохраняется

Все в `UserDefaults.standard` (встроенное хранилище macOS)

```swift
UserDefaults.standard.set(currentLanguage.rawValue, forKey: "selectedLanguage")
UserDefaults.standard.set(useIPLocation, forKey: "useIPLocation")
```

### При перезагрузке

При следующем запуске приложение:
- Загружает сохраненный язык
- Загружает сохраненный выбор GPS/IP
- Все восстанавливается автоматически

## 🏗️ Архитектура локализации

### Файл: Localization.swift

```swift
enum Language: String, CaseIterable {
    case english = "en"
    case russian = "ru"
    case polish = "pl"
    case belarusian = "be"
}

class LocalizationManager: ObservableObject {
    @Published var currentLanguage: Language
    
    func localize(_ key: String) -> String {
        // Переводит ключ на текущий язык
    }
}
```

### Использование

```swift
// В ContentView
@ObservedObject private var localization = LocalizationManager.shared

// Использование
Text(localization.localize("updated"))
Text(localization.localize("clear"))
```

### Добавление нового перевода

1. Добавьте ключ в `translationDictionary()`:
```swift
"my_new_key": "Мой новый ключ"
```

2. Используйте в коде:
```swift
Text(localization.localize("my_new_key"))
```

## 🔄 Обновление места

### Автоматическое обновление

Место обновляется при:
- Первом запуске приложения
- Изменении GPS координат (если включена)
- Переключении между GPS и IP
- Нажатии кнопки "Обновить"

### Процесс обновления

**GPS**:
1. Запрашивает координаты у CLLocationManager
2. Использует CLGeocoder для reverse geocoding
3. Получает город/страну
4. Отображает результат

**IP**:
1. Запрашивает данные у ipapi.co
2. Парсит JSON ответ
3. Извлекает город и страну
4. Отображает результат
