#### 0. **About Dialog (Новое в v2.0.2)**

#### 1. **Определение местоположения GPS/IP**
- Автоматический выбор между GPS и IP
- Переключатель в интерфейсе приложения
- Сохранение выбора пользователя
- Fallback на IP, если GPS недоступна

#### 2. **Прогноз на 5 дней**
- Карточки для каждого дня недели
- Макс/мин температура
- Описание погодных условий
- Удобный формат даты

#### 3. **Уровень активности (0-100%)**
- Процентная шкала вместо дискретных вариантов
- Цветовое кодирование:
  - 🟢 80-100% — Идеально
  - 🔵 60-79% — Хорошо
  - 🟠 40-59% — Допустимо
  - 🔴 0-39% — Неподходящие условия
- Прозрачный алгоритм расчета

#### 4. **Многоязычная поддержка**
- Английский (English)
- Русский (Русский)
- Польский (Polski)
- Беларусский (Беларусь)
- Сохранение предпочтения языка

#### 5. **Оптимизированный интерфейс**
- Вертикальная ориентация (400x700px)
- ScrollView для большого контента
- Красивые иконки и emoji
- Минималистичный дизайн

## 🔧 Технические улучшения

### Обновленные компоненты

**Models.swift**
- Новая структура `ActivityScore` для расчета
- Поддержка `DailyForecast` для прогноза
- Улучшенный алгоритм взвешивания (Т:40%, П:40%, В:20%)

**LocationManager.swift**
- Интеграция с ipapi.co для IP геопозиции
- CLGeocoder для обратного кодирования (координаты → название города)
- Сохранение в UserDefaults

**WeatherService.swift**
- Запросы к Open-Meteo с параметром `daily`
- Кэширование результатов
- Улучшенная обработка ошибок

**ContentView.swift** (полная переработка)
- Новые представления:
  - `currentWeatherView` — текущая погода
  - `scoreView` — шкала активности
  - `forecastView` — прогноз 5 дней
  - `controlsView` — управление (GPS/IP, язык)
- ScrollView для защиты от обрезания

**Localization.swift**
- Поддержка 4 языков (en, ru, pl, be)
- Использование словарей вместо enum
- Динамическое переключение языков

## 📥 Инсталляционные пакеты

### Доступные форматы

1. **WalkAdvisor_2.0.1.dmg** (рекомендуется)
   - Образ диска для всех пользователей
   - Удобная установка перетаскиванием
   - ~2.8 MB

2. **WalkAdvisor_2.0.1_arm64.pkg**
   - Пакет для Apple Silicon (M1, M2, M3...)
   - Может быть распространен через App Store (если будет)
   - ~1.2 MB

3. **WalkAdvisor_2.0.1_x86_64.pkg**
   - Пакет для Intel Mac
   - Может быть распространен через App Store (если будет)
   - ~1.2 MB

### Установка

```bash
# Из DMG образа
open dist/WalkAdvisor_2.0.1.dmg
# Перетащите Walk Advisor.app в Applications

# Или из PKG напрямую
installer -pkg dist/WalkAdvisor_2.0.1_arm64.pkg -target /
```

## 📞 Поддержка

### Проблемы с установкой
1. Убедитесь, что у вас macOS 12.0 или выше
2. Проверьте свободное место на диске (~10 MB)
3. Убедитесь, что приложение не повреждено (файл MD5)

### Проблемы с геопозицией
```bash
# Дайте разрешение на GPS
tccutil grant location /Applications/Xcode.app

# Или вручную: System Settings → Privacy & Security → Location Services
```

### Проблемы с прогнозом
- Проверьте интернет соединение
- Убедитесь, что api.open-meteo.com доступен
- Попробуйте переключить GPS/IP
