INSTALLATION OPTIONS
================================================================================

1. USING DMG (Recommended for users)
   - Download: Walk Advisor_2.0.1.dmg
   - Double-click to mount
   - Drag "Walk Advisor" to Applications
   - Run from Applications folder

2. USING PKG (For scripts/automation)
   - Apple Silicon: Walk Advisor_2.0.1_arm64.pkg
   - Intel: Walk Advisor_2.0.1_x86_64.pkg
   - Command: installer -pkg file.pkg -target /

3. FROM SOURCE
   - Clone repository
   - cd WeatherApp
   - ./build.sh
   - open build/Walk\ Advisor.app

================================================================================
QUICK START
================================================================================

1. LAUNCH APP
   - Open Applications folder
   - Double-click "Walk Advisor"
   - Allow location access

2. GRANT PERMISSIONS
   - System will ask for location access
   - Grant location permission
   - App starts automatically fetching weather

3. USE THE APP
   - View current weather and activity level
   - See 5-day forecast
   - Toggle between GPS and IP location
   - Change language in menu

4. FIRST RUN
   - App shows current location (GPS or IP)
   - Fetches weather from Open-Meteo
   - Calculates activity recommendation
   - Displays forecast and activity level

KNOWN ISSUES & SOLUTIONS
================================================================================

ISSUE: GPS not working
SOLUTION: Grant location permission in System Settings → Privacy & Security

ISSUE: Wrong location from IP
SOLUTION: IP geolocation is city-level; use GPS for accuracy

ISSUE: Weather not loading
SOLUTION: Check internet connection; api.open-meteo.com must be accessible

ISSUE: App crashes on launch
SOLUTION: Ensure macOS 12.0+; try clean rebuild
