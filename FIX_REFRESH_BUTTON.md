# Исправление: Кнопка "Обновить" теперь учитывает режим GPS/IP

**Дата**: 2026-05-22
**Версия**: 2.0.1

---

## 📋 Проблема

При нажатии на кнопку "Обновить" приложение не давало четкой визуальной обратной связи о том, что происходит загрузка, и кнопка оставалась активной даже во время загрузки.

Кроме того, логика загрузки могла быть не очень понятной для пользователя в отношении того, какой режим (GPS или IP) используется.

---

## ✅ Решение

### 1. **Визуальная обратная связь при загрузке** 🔴 → ✅

**Было**:
```swift
Button(action: {
    Task { await loadWeather() }
}) {
    HStack(spacing: 6) {
        Image(systemName: "arrow.clockwise")
        Text(localization.localize("refresh"))
    }
    // Кнопка всегда активна
}
```

**Стало**:
```swift
Button(action: {
    Task { await loadWeather() }
}) {
    HStack(spacing: 6) {
        // Иконка вращается во время загрузки
        Image(systemName: isLoading ? "arrow.clockwise.circle.fill" : "arrow.clockwise")
            .rotationEffect(.degrees(isLoading ? 360 : 0))
            .animation(.linear(duration: 1).repeatForever(autoreverses: false), value: isLoading)
        Text(localization.localize("refresh"))
    }
    .background(Color.white.opacity(isLoading ? 0.3 : 0.2))  // Темнеет при загрузке
}
.disabled(isLoading)  // Отключается при загрузке
```

**Эффект**:
- ✅ Иконка вращается при загрузке
- ✅ Фон кнопки темнеет при загрузке
- ✅ Кнопка отключается (disabled) во время загрузки
- ✅ Пользователь видит, что происходит процесс обновления

---

### 2. **Улучшена логика определения режима** 

**Было**:
```swift
if locationManager.useIPLocation {
    // загружаем IP
} else if let location = locationManager.location {
    // используем GPS
} else {
    // показываем ошибку
}
```

**Стало** (с комментариями):
```swift
if locationManager.useIPLocation {
    // IP режим: загружаем IP координаты если нужно
    if let ipLoc = locationManager.ipLocation {
        // используем существующие
    } else {
        // загружаем свежие
    }
} else if let location = locationManager.location {
    // GPS режим: используем GPS координаты
} else {
    // Нет доступных координат
    if !locationManager.isAuthorized && !locationManager.useIPLocation {
        showIPPrompt = true
    } else {
        errorMessage = localization.localize("no_location")
    }
}
```

**Улучшения**:
- ✅ Четкое разделение логики GPS и IP режимов
- ✅ Правильная обработка ошибок для каждого режима
- ✅ IP координаты загружаются если они не были загружены ранее
- ✅ Понятные сообщения об ошибках в каждом случае
