## 🎯 Использование приложения

### Первый запуск

1. Приложение запросит доступ к GPS
2. **Разрешите** или **используйте IP**
3. Смотрите текущую погоду и прогноз

### Ежедневное использование

1. Смотрите **уровень активности %**
2. Планируйте прогулки по **прогнозу 5 дней**
3. Переключайте **GPS ↔ IP** если нужно
4. Нажимайте **Обновить** для свежих данных

### Сохранение выбора

- Ваш выбор GPS/IP **сохраняется автоматически**
- При следующем запуске будет использоваться сохраненный выбор
- Можно менять в любой момент
