## 📌 Что это за приложение?

**Walk Advisor** — нативное приложение для macOS, которое помогает решить:
- **Идти ли мне гулять сегодня?** 🚶
- **Подходит ли погода для бега?** 🏃
- **Как будет погода в ближайшие дни?** 📅

### Основные возможности

- 🌍 **Определение местоположения**: GPS или IP
- 🌤️ **Текущая погода**: температура, влажность, ветер
- 📊 **Уровень активности**: процентная шкала (0-100%)
- 📅 **Прогноз на 5 дней**: макс/мин температуры
- 💾 **Запоминание выбора**: GPS/IP переключатель
- 🎨 **Красивый интерфейс**: вертикальное окно 400x700

## 🎛️ Как использовать

### Определение местоположения

1. **Первый запуск**: Приложение попросит доступ к GPS
   - **Разрешить**: Будет использовать точные координаты
   - **Использовать IP**: Определит по IP-адресу (менее точно)

2. **Дневное использование**: Переключайте GPS/IP в интерфейсе
   - Выбор сохранится автоматически
   - При следующем запуске будет использоваться сохраненный выбор

### Уровень активности

Шкала от 0 до 100%:
- **🟢 80-100%**: Идеально! Идите гулять/бегите
- **🔵 60-79%**: Хорошо для прогулки
- **🟠 40-59%**: Допустимо, но не очень
- **🔴 0-39%**: Лучше остаться дома

### Планирование

Смотрите прогноз на 5 дней и выбирайте лучший день для прогулки!

## ⚙️ Первая установка

### Шаг 1: Разрешить геопозицию

При первом запуске приложение попросит разрешение:

```
"Walk Advisor" хочет получить доступ к вашему местоположению
```

Выберите:
- **Разрешить** - для точных координат (рекомендуется)
- **Не разрешать** - будет использовано IP определение

### Шаг 2: Дождитесь загрузки

Приложение загрузит текущую погоду (~1 сек)

### Шаг 3: Наслаждайтесь!

Смотрите погоду и рекомендацию.

## 💡 Советы

1. **GPS медленный?** 
   → Выберите IP определение (быстрее, менее точно)

2. **Экран слишком маленький?**
   → Перетащите край окна, чтобы увеличить
   → Или закройте прогноз/скролл вверх

3. **Не видно всей информации?**
   → Используйте ScrollView (скролл внутри окна)

4. **Хотите изменить цвета?**
   → Отредактируйте ContentView.swift
   → Пересоберите ./build.sh

## 🐛 Устранение неполадок

### "Геопозиция не работает"

```bash
# Выдайте разрешение Xcode
tccutil grant location /Applications/Xcode.app

# Или вручную:
# Параметры → Приватность и безопасность → Геопозиция
# → включить терминал/Python
```

### "API недоступен"

- Проверьте интернет соединение
- Убедитесь, что https://api.open-meteo.com доступен

### "Приложение не компилируется"

```bash
# Обновить Command Line Tools
xcode-select --install

# Пересобрать
rm -rf build/
./build.sh
```
