## 📥 Инсталляционные пакеты

### Доступные форматы

1. **Walk Advisor_2.0.2.dmg** (рекомендуется)
   - Образ диска для всех пользователей
   - Удобная установка перетаскиванием
   - ~1.5 MB
   - Архитектура: Intel (текущая система)

2. **Walk Advisor_2.0.2_arm64.pkg**
   - Пакет для Apple Silicon (M1, M2, M3...)
   - Может быть распространен через App Store (если будет)
   - ~1.2 MB

3. **Walk Advisor_2.0.2_x86_64.pkg**
   - Пакет для Intel Mac
   - Может быть распространен через App Store (если будет)
   - ~1.2 MB

### Установка

```bash
# Из DMG образа
open dist/Walk\ Advisor_2.0.2.dmg
# Перетащите Walk Advisor.app в Applications

# Или из PKG напрямую
installer -pkg dist/Walk\ Advisor_2.0.2_arm64.pkg -target /
installer -pkg dist/Walk\ Advisor_2.0.2_x86_64.pkg -target /
```
